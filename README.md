# Front-End Project of Student Information System

### For Installation
```
yarn install
```

For Execution
------------------

### LOCAL API
```
yarn build:local
```
```
yarn dev:local
```

### LIVE API
```
yarn build
```
```
yarn dev
```

Local URL : http://localhost:8586/
------------------

Live URL : https://sis-fe.herokuapp.com/
